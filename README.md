# Портфолио из учебных проектов

Репозиторий содержит проекты, выполненные мною в рамках онлайн обучения в образовательном сервисе "Яндекс.Практикум". Профессия: Специалист по Data Science.

Чтобы удержать клиентов и привлечь новых, сделать бренд более узнаваемым, команда сервиса проводит исследования аудитории, и публикует интересные результаты. Как музыка, которая звучит по дороге на работу в понедельник утром, отличается от той, что играет в среду или в конце рабочей недели? Имеются данные для Москвы и Петербурга. Необходимо сравнить, что и в каком режиме слушают их жители. | pandas |

Для корректировки рекламного бюджета, коммерческому департаменту нужна информация о том, какой тариф приносит больше денег. | pandas, matplotlib, numpy |

| [06_Введение в машинное обучение](https://github.com/IvanZhoglik/yandex-practicum-projects/tree/main/06_introduction_to_machine_learning) | Рекомендация тарифов | На основе данных о поведении клиентов оператора мобильной связи требуется построить модель для задачи классификации, которая выберет подходящий тариф. | pandas, matplotlib, sklearn |
| [07_Обучение с учителем](https://github.com/IvanZhoglik/yandex-practicum-projects/tree/main/07_supervised_learning) | Отток клиентов | Из банка стали уходить клиенты. Предоставлены исторические данные о поведении клиентов и расторжении договоров с банком. | pandas, matplotlib, numpy, sklearn |
| [08_Машинное обучение в бизнесе](https://github.com/IvanZhoglik/yandex-practicum-projects/tree/main/08_machine_learning_in_business) | Выбор локации для скважины | Для нефтедобывающей компании нужно решить, где бурить новую скважину. | [09_Сборный проект-2](https://github.com/IvanZhoglik/yandex-practicum-projects/tree/main/09_integrated_project_2) | Сборный проект-2. Восстановление золота из руды | Для компании, разрабатывающей решения для эффективной работы промышленных предприятий, требуется разработать модель машиннного обучения. Можель должна предсказать коэффициент восстановления золота из золотосодержащей руды. | pandas, matplotlib, seaborn, numpy, sklearn |
| [10_Линейная алгебра](https://github.com/IvanZhoglik/yandex-practicum-projects/tree/main/10_linear_algebra) | Защита персональных данных клиентов | Для страховой компании необходимо разработать метод защиты данных клиентов. Нужно защитить данные, чтобы при преобразовании качество моделей машинного обучения не ухудшилось. Подбирать наилучшую модель не требуется. | pandas, matplotlib, seaborn, numpy, sklearn |
| [11_Численные методы](https://github.com/IvanZhoglik/yandex-practicum-projects/tree/main/11_numerical_methods) | Определение стоимости автомобилей | Сервис по продаже автомобилей с пробегом разрабатывает приложение для привлечения новых клиентов. Предосталвены исторические данные: технические характеристики, комплектации и цены автомобилей. | pandas, matplotlib, seaborn, numpy, sklearn, catboost, lightgbm, xgboost, time |
